# roq-java-samples

Example projects demonstrating how to use Roq's Java bindings.


## Prerequisites

You should run this from a conda environment having the `roq-java` package installed.


## Using

```bash
java \
  -Djava.library.path=$CONDA_PREFIX/lib \
  -jar roq-samples.jar \
  $HOME/run/deribit.sock
```
